ID Design System

Creating a White Label Design System for multiple clients

Client

Confidential

Location

Brazil

Year

2023 - 2024

I worked throughout 2023 and 2024 building the Design System for a company.

During this project, we structured and standardized key elements from the very beginning. This included design tokens, components, templates, utilities for files, and more.

Additionally, the design system needed to support different modes for various clients, resulting in a highly versatile system with numerous possibilities for screen creation.

Design tokens

Design tokens were developed to standardize the interface elements.

Components

Various components were developed for use in the interfaces. Along with this development, a handoff was created with all the tokens used in each of them.

Templates

A standard artboard was created with predefined paddings, auto layouts, and replaceable elements. This allows components to be simply dragged and dropped, automatically fitting into the screen.
